About HECinBox

An automated 2D HEC-RAS flood simulation pipeline, built to remove the manual steps between a gage reading and an inundation map.

The project

HECinBox packages a full HEC-RAS 2D unsteady workflow into a single container. It pulls real-time USGS discharge and NOAA tide records, injects them into a model as boundary conditions, runs the HEC-RAS Linux engine headless, then returns depth maps, animations, validation statistics, and threshold alerts. It is built for engineers and research groups that need to run the same model repeatedly without touching it.

HEC-RAS is developed by the US Army Corps of Engineers Hydrologic Engineering Center. HECinBox is an independent product and is not affiliated with or endorsed by USACE.
