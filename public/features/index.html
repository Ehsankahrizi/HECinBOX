Features

What is in the box

Everything the workflow needs, from data intake to the alert email.

Live boundary conditions

USGS discharge and stage, NOAA tides, National Water Model and STOFS forecasts. Fetched by API and written into the plan as flow, stage, or normal depth.

Headless HEC-RAS engine

The Linux solver runs inside the container. Pick the thread count, close the tab, the run keeps going. Models can be read from S3, R2, MinIO, or Wasabi.

Inundation mapping and video

Filled mesh cells on the GPU, depth class bands, shaded relief, satellite basemaps, and a 3D view. Every time step also becomes a playable flood animation.

Validation built in

Modeled series compared against the observed gage record with NSE, RMSE, MAE, bias, and a one to one scatter.

Alerts and unattended runs

Rules on cell depth, domain peak, or wetted area send an email the moment a threshold is crossed. Set an interval and the pipeline repeats on its own.

Rain on grid

Direct precipitation on the mesh from a constant rate, an AORC or HRRR gridded product, or an uploaded DSS file.
